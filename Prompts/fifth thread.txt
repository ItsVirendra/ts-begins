2. Learned that:

   * TypeScript interfaces exist only at compile time.
   * Runtime JavaScript cannot import interfaces as values.
   * Type-only imports should use:

     import type { User } from "...";

3. Learned that:

   * Writing TypeScript code and running TypeScript code are different skills.
   * Project configuration can fail even when application logic is correct.

4. Learned the standard TypeScript workflow:

   TypeScript (.ts)
   ↓
   tsc
   ↓
   JavaScript (.js)
   ↓
   node

5. Learned basic TypeScript project setup:

   npm init -y
   npm install -D typescript
   npx tsc --init

6. Encountered and fixed module configuration issues:
